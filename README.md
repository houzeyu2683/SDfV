# SDfV
Speaker Detection from Video

紀錄時間：2023/10/20

該專案主要收集視頻類型的資料，資料中需要有人物面對鏡頭說話的部份，對聲音與畫面的同步有一定的要求。

已完成：

- [【數位主播午報】][1]：影片開頭會有固定的開場白；

- [【數位主播晚報】][2]：影片開頭會有固定的開場白；

- [中視新聞全球報導][3]：影片開頭會有固定的開場白，部份下載且不繼續，原因一言難盡； 

進行中：

- [【主播焦點新聞】東森新聞][5]：影片開頭不會有固定開場，主播有時候會走動；
